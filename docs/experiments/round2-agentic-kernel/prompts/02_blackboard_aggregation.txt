Use the execute_python tool to run Python code that:
1. Spawns 2 workers into "chat" — one asks "What is 2+2? Reply with just the number." and one asks "What is 3+3? Reply with just the number."
2. Polls until both complete
3. Writes each result to the blackboard with blackboard_write()
4. Reads them back with blackboard_read() and writes a summary
5. Prints the blackboard keys and summary

Available functions (pre-loaded, no imports): spawn_agent(room, prompt), agent_status(handle), get_result(handle), sleep(ms), blackboard_write(key, value), blackboard_read(key), blackboard_keys(), print()
